БЫСТРЫЙ СТАРТ
=============

1. Перейти в папку:
   cd mlclass-ex2

2. Показать справку:
   make help

3. Запустить основные скрипты:
   make ex2              # простая регрессия
   make ex2_reg          # с регуляризацией

4. Тестирование:
   make test_lambda      # разные lambda
   make test_degree      # разные degree
   make test_without_reg # без регуляризации

5. Всё сразу:
   make all

Графики показываются автоматически!
